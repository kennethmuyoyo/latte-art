Latte Art Generation

User chooses pattern to practice. App shows arrow to follow using CoreGraphics and Vision. Metalkit simulates the pattern formed from the arrows.

Create an app that utilize [Physics Engine, MetalKit, CoreGraphics, Vision, ARKit] to [Calculate physical movement, simulate & render latte art, and showcases pattern to follow for our user]

This is the flow of the app: 
We are creating a Swift app that will have this flow, user gets instructions on how to setup their equipments, the jug, the cup and the tripod. The user is also told to fill the jug with water.
The user then selects a pattern to create between tulips, heart and rosetta.
Then next the user is shown a pattern on the water surface which shows how to fill the water into the cup and this will show a simulation using Metal framework on the Phone as the user pours the water into the cup, then we should see the position that the user pours the water into the cup and tell them if they are following the pattern or not.
Then when the cup is full and the simulation shows that the coffee is ready for the foam we now show directions to form the selected pattern they want to create. So we still show the simulation of what the pour is making to show the pattern that the pour is making and if the user misses the instructions we show a visual indicator that they have missed, and if they are on track we show them what they are crating.
User should pour at certain points and in a certain direction to form the patterm.
